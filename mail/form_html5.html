Description
A newsletter opt-in form using jQuery, PHP and mySQL.
An alternative version using html5 for validation is included in this package.
This is a newsletter sign-up for your web site so visitors can subscribe to your newsletter.
The visitor is required to enter a name and a valid email.
Validation is done inline without reloading the page and redirecting the user.
The "Thank you" message is also displayed inline (try it above).
Note: an alternative version of this page that uses html5 validation is included in this package.
In the background, the code in this package gives you two options:
  1. Save the name and email to a txt file. Values are enclosed by double quotes and are comma separated so you can easily import this file into other applications. Example: "youremail@domain.com", "Your Name"
  2. Save to a mySQL database. First it checks if the email exists already and if not it is added. Sample script to create the table is included. Complete and well commented source code included.
